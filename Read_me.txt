Котошифр - это программа шифровальщик, которая шифрует текст.
Идея следующая. Через каждую букву незашифрованного текста вставляется слово кот.
Пробелы, точки и запятые игнорируются.

Также программа умеет считывать данные из текстовых файлов. То есть файл должен быть
с расширением txt.

______________________________________________________________________________________

***в четвертом commit добавлена следующая возможность***
Теперь программа может записывать шифрованный текст в текстовый файл.
Недостаток состоит в том что путь на чтение и запись жестко прописан,
т.е. пользователь не может сам выбирать, какой он хочет зашифровать текстовый файл.
И не может указать дирректорию на сохранение зашифровнного файла.

***В пятом commit добавлены следующие возможности***
Теперь пользователь может выбирать любой текстовый файл с помощью вызова стандартного
диалога на выбор файлов для его шифрования "котошифром".
Также после шифрования пользователь может сохранить полученный шифр в текстовом файле
в любой директории, так как сохранение происходит с помощью вызова стандартного диалога
на сохранение файла.

***В шестом commit добавлены следующие возможности***
Теперь программа не шифрует перенос каретки на новую строку, т.е.
при шифровании многострочного текста переход на новую строку не шифруется.
Также добавлена проверка на корректное указание имени текстового файла,
куда сохраняются зашифрованные данные, т.е. имя текстового файла не может быть пустым.
